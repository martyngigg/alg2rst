Description
-----------

Performs beam flux correction for TOF SANS data.

The correction goes as follows:

        :math:`I({\lambda}) = I_0({\lambda}) / \Phi_{sample}`

where

        :math:`\Phi_{sample} = \frac{M_{sample}}{M_{ref}} \Phi_{ref}`
