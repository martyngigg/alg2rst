Description
-----------

Loads an LLB MIBEMOL TOF NeXus file into a `Workspace2D <Workspace2D>`__
with the given name.

This loader calculates the elastic peak position (EPP) on the fly.

To date this algorithm only supports the MIBEMOL instrument.
